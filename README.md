# ToDo Web App

> The main goal of this exercise is to develop a web application with __Ruby on Rails__ in which we are going to implement a web system for managing a ToDo Web Application similar to Todoist.

## Built With

- Ruby 2.7.0
- Rails 6.0.2.1
- Bootstrap 4
